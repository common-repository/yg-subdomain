YG-Subdomains allows users to setup there specified page as subdomains. It's a lite and fast code.

== Description ==

- On "add post" page there will be a checkbox, "Fake Subdomain"
- When checked it uses a custom field called fake_sub to create a fake subdomain for the post



== Installation ==


Manual Installation Subdomains plugin:

    Download and unzip the file

    Upload the full extracted folder to the /wp-content/plugins/ directory

    Activate the plugin through the 'Plugins' menu in WordPress.

You're done! The Plugin ready to use according to the directions in the description.
